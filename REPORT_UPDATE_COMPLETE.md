## 📄 PDF 출력 최적화

### 1. 파일명 변경
- **이전**: `G-DAX_진단리포트_{ID}_{날짜}.pdf`
- **현재**: `산업일자리전환_사전진단보고서_{ID}_{날짜}.pdf`

### 2. 2페이지 출력 최적화
**PDF 생성 옵션 개선**:
```javascript
{
  margin: [8, 8, 8, 8],        // 여백 축소 (10mm → 8mm)
  scale: 1.5,                  // 해상도 최적화 (2배 → 1.5배)
  windowWidth: 1200,           // 렌더링 너비 고정
  pagebreak: {
    mode: ['avoid-all', 'css'],  // CSS 페이지 브레이크 우선
    avoid: ['.no-break', 'canvas']  // 차트 페이지 나누기 방지
  }
}
```

**레이아웃 압축**:
- ✅ 섹션 패딩: 8 → 4 (50% 축소)
- ✅ 마진: 8 → 4 (50% 축소)
- ✅ 차트 높이: 500px → 350px (30% 축소)
- ✅ 제목 크기: text-2xl → text-xl
- ✅ 본문 크기: 최적화

**CSS 페이지 제어**:
```css
.page-break-before { page-break-before: always; }  /* 페이지 시작 */
.page-break-after { page-break-after: always; }    /* 페이지 끝 */
.no-break { page-break-inside: avoid; }            /* 나누기 방지 */
```

### 3. 인쇄 최적화
```css
@media print {
  .no-print { display: none !important; }  /* 버튼 숨김 */
  body { margin: 0; padding: 0; }          /* 여백 제거 */
  .shadow-lg { box-shadow: none; }         /* 그림자 제거 */
}
```

## 🚀 사용 방법

### 1. 웹에서 보기
```
https://g-dex-survey.pages.dev/report/{설문ID}
```

### 2. PDF 다운로드
1. 리포트 페이지 접속
2. 하단 **"PDF 다운로드"** 버튼 클릭
3. 자동으로 다운로드 (파일명: `산업일자리전환_사전진단보고서_{ID}_{날짜}.pdf`)

### 3. 인쇄하기
1. 리포트 페이지 접속
2. 하단 **"인쇄하기"** 버튼 클릭
3. 브라우저 인쇄 대화상자에서 "PDF로 저장" 선택
4. 출력: 2페이지 (자동 최적화)
